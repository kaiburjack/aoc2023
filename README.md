How to run?

First, you need to install Go for your system.

Then, to run the solution for any day and part,
`cd` into the day and part folder and execute
`go run main.go` to compile and run.
Or run `go build -o main main.go` to compile
and then `./main` to run the executable.

The respective input for this day and part is
contained in the folder.
